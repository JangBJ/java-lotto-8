# 로또 
## 개요
간단한 로또 발매기를 구현한다.

## 요구사항
1. 로또 번호의 숫자 범위는 1~45이다.
   1. 중복은 허용하지 않는다.
2. 사용자의 입력으로 6개의 중복되지 않는 숫자(당첨 번호)를 입력받는다.(구분자는 "," 입니다)
3. 사용자의 입력으로 보너스 숫자를 1개 입력 받는다.
4. 당첨은 1~5등까지 존재한다.
   1. 1등: 6개의 번호 일치
   2. 2등: 5개 + 보너스 번호
   3. 3등: 5개 번호 일치
   4. 4등: 4개 번호 일치
   5. 5등: 3개 번호 일치
6. 사용자는 로또 구입 금액을 입력하고 구입 금액에 해당하는 로또 갯수를 발행한다.
   1. 1장의 가격은 1000원이다.비교
7. 생성된 로또들의 번호들을 입력받은 당첨 번호와 비교후 해당 결과를 반환합니다.
8. 사용자의 잘못된 입력은 `IllegalArgumentException`을 발생시키고 "[ERROR]"로 시작하는 에러 메시지 출력 후 다시 입력 받습니다.

## 데이터 검증 목록
### 금액
1. 금액은 어느정도까지 받을 수 있는가?
2. 금액이 1000단위로 떨어지는가
3. 숫자인가?

### 당첨 번호
1. 숫자인가?
2. ,외의 다른 특수문자가 있는가?
3. 6개인가?(미만이나 초과하진 않았는가)
4. 1이상 45이하의 범위를 만족하는가?
5. 중복이 있는가?

### 보너스 번호
1. 숫자인가?(문자, 특수문자가 아닌지)
2. 1~45 범위의 숫자인가?
